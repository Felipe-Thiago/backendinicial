# Primeiro Projeto backend
>> Banco de dados Não Relacional

## Introdução
Execute os seguintes comandos:
```bash
npm init
npm i express
```
Edite o arquivo package.json com o seguinte conteúdo:
```json
    "main": "api/index.js",
    "type": "module",
    "engines": {
        "node": "18.x",
        "npm": "9.x"
    }
```
